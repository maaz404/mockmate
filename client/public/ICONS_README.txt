Place the following files in this folder to brand the favicon/app icons:

- favicon.ico           (16/32/48/64 sizes embedded)
- favicon-16.png        (16x16 PNG)
- favicon-32.png        (32x32 PNG)
- apple-touch-icon.png  (180x180 PNG)
- icon-192.png          (192x192 PNG for PWA)
- icon-512.png          (512x512 PNG for PWA)

You can export these from src/assets/mockmate-logo.png using an image editor
or a favicon generator. Create React App will reference these during build.
